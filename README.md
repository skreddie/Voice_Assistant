#  Python Voice Assistant (Alexa-like)

This project is a simple **voice-controlled assistant** built using Python.  
It listens to voice commands, responds using speech, plays songs on YouTube, tells the time, searches Wikipedia, and tells jokes.

---

##  Features

-  Voice input using microphone
-  Text-to-speech responses
-  Play songs on YouTube
-  Tell current time
-  Wikipedia search
-  Joke generator
-  Basic conversational replies

---

##  Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- pywhatkit
- wikipedia
- pyjokes
- datetime

---

##  Installation

1. Clone the repository

2. Install required libraries
   
   
>  Note:  
> On Windows, `pyaudio` may require a precompiled wheel.

---

##  How to Run

Make sure:
- Microphone is connected
- Internet connection is active

---

##  Example Voice Commands

| Command | Action |
|-------|-------|
| play despacito | Plays song on YouTube |
| what is the time | Tells current time |
| who the heck is Elon Musk | Wikipedia summary |
| tell me a joke | Tells a joke |
| are you single | Fun reply |
| date | Predefined response |

Say **"Alexa"** before giving a command.

---

##  How It Works

1. Listens to voice input
2. Converts speech to text
3. Matches command
4. Executes action
5. Responds using speech
